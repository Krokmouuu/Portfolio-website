# bleroy.dev — Portfolio

Portfolio personnel de **Benoit L.**, fondateur & développeur Full Stack.

Le site présente mon profil, mes compétences, mon parcours et une sélection de projets.

**Live :** https://bleroy.dev

## Ce que tu trouveras sur le site

- Une landing “impact” (Hero) avec CTA vers les projets.
- Une section **À propos** (présentation + stats).
- **Skills** (stack principale + skills secondaires + outils).
- **Experience** sous forme de timeline.
- **Mes Projets** (cards avec liens GitHub / Live, et App Store pour Arc Cycle).
- **Services** (ce que je peux proposer).
- **Contact** (GitHub, LinkedIn, Malt, email).

## Tech & design

- **React + TypeScript**
- **Vite**
- **Tailwind CSS**
- **motion** (animations)
- **lucide-react** (icônes)
- Fond animé en canvas (particules)
